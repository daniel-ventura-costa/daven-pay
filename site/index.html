Como instalar o projeto

Siga os passos abaixo, não se preocupe, é bem facil ;)

Hosts

Abra o arquivo hosts do seu sistema e adicione os registros abaixo no fim do arquivo:

  • 127.0.0.1 davenpay.dev
  • 127.0.0.1 api.davenpay.dev
Docker

Precisamos iniciar a infraestrutura, navegue até a pasta do projeto e execute os comandos abaixo:

  • docker-compose up -d --build
  • composer install
Popular banco de dados

Agora precisamos criar as tabelas e popular alguns dados no nosso banco de dados:

  • docker exec -it api01 php artisan migrate:refresh
  • docker exec -it api01 php artisan db:seed

Modelagem do banco de dados

Infraestrutura desenvolvida para este projeto

Esta infra foi pensada para ser desacoplada e escalável horizontalmente.

  • Utilizar um microframework para a API rest, o LUMEN foi o escolhido.
  • Foi escolhido o banco de dados em memória REDIS para gerenciar as transações em background.

Como funciona o processo de transferência

Load Balancer (NGINX)

Utilizamos um balanceador de carga (NGINX) para as chamadas feitas na API

API em Lumen

Utilizamos o microframework Lumen para as requisições

Manda para o REDIS

Guarda a transação no REDIS, para ser processado em "fila"

Processador de Jobs

Uma instância que vai processar as transferências fica "escutando" os registros do REDIS

Pontos otimizados

Proposta de melhoria na arquitetura

  • Autenticação por token JWT
  • Payload foi trocado o ID por um UUID (hash da carteira)
  • Para comportar o soft delete foi adicionado o campo deleted_at no banco de dados
